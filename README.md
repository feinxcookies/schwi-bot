# Schwi Bot

A bot made by [feinxcookies](https://github.com/feinxcookies)

## How to run
0. Install `npm`
1. Run `npm install discord.js`
2. Ask `feinxcookies` for the `.env` file. it is used with the `dotenv` package to pass in environment variables
3. Run `node bot.js`
